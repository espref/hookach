# Hookach – подбор табака и миксов для кальяна

Приложение для кальянной: гость сканирует QR-код, выбирает вкусы и создает микс, отправляет заказ по номеру стола. Администратор (кальянщик) принимает заказы и меняет статусы приготовления.

## Запуск проекта

1. Установите зависимости:

```bash
npm install
```

2. Запустите сервер:

```bash
npm start
```

По умолчанию сервер поднимется на `http://localhost:3000`.

## Данные для входа в админку

- **Логин**: `admin`
- **Пароль**: `admin12345689`

Скрытая кнопка входа расположена в футере сайта (в правой части). Нажмите по малозаметному тексту в футере, чтобы открыть форму входа администратора.
